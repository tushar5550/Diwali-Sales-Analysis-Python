# Diwali Sales Analysis using Python

# Project Overview
This project performs Exploratory Data Analysis (EDA) on a real-world Diwali Sales dataset using Python. The objective is to analyze customer purchasing behavior, identify sales trends, and generate business insights that can help improve marketing strategies and customer targeting.
The project includes data cleaning, preprocessing, visualization, and business insight generation using Python data analysis libraries.

# Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

# Objectives
- Perform data cleaning and preprocessing.
- Analyze customer purchasing behavior.
- Identify high-performing states, occupations, and product categories.
- Explore customer demographics such as gender and age groups.
- Generate meaningful business insights through data visualization.
- Support data-driven business decision making.

# Dataset Information
- Dataset Name: Diwali Sales Data
- File Format: CSV
- Records: 11,000+ customer purchase records
- Domain: Retail Sales
- Attributes: Gender, Age Group, State, Marital Status, Occupation, Product Category, Orders and Sales Amount.

# Project Workflow
1. Imported the required Python libraries.
2. Loaded the dataset using Pandas.
3. Performed data cleaning by removing unnecessary columns and handling missing values.
4. Conducted Exploratory Data Analysis (EDA).
5. Created visualizations to identify customer purchasing trends.
6. Derived business insights based on the analysis.

# 📈 Key Business Insights
- Married women in the age group of 26–35 years contributed the highest number of purchases.
- Customers from Uttar Pradesh, Maharashtra, and Karnataka generated the highest sales.
- Professionals working in the IT, Healthcare, and Aviation sectors showed strong purchasing behavior.
- Food, Clothing, and Electronics were among the top-selling product categories.
- Businesses can improve sales by targeting married women aged 26–35 through personalized marketing campaigns and regional promotions.
